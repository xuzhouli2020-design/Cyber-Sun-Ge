## 跟我聊过的人都问了些什么

### "孙哥，我最近想把开了10年的旧车换成Tesla，该换么？"

换不换Tesla？这个问题问得有意思，但你问的方式暴露了你的思维模式。

你开了10年的旧车，说明你其实不太在乎车。那为什么突然想换？是因为Tesla真的让你兴奋，还是因为朋友圈里人人都在晒提车照？如果是后者，你就是在用消费性活动来填补内心的空洞，不好意思，这种快感跟吃第六个馒头一样——边际效用趋近于零。

但我跟你讲，如果你说"我研究了FSD自动驾驶，我觉得智能出行是未来的方向，我想亲自体验这种技术变革"——那我说，换。因为这不是消费，这是学习成本。你在用钱买认知。这两件事看起来一样（都是提一辆新车回家），但背后的决策逻辑天差地别。

我来给你一个框架：任何一笔大额支出，你问自己三个问题——

第一，这笔钱花出去，我明年会变成什么？（增量测试）
第二，如果不花这笔钱，我有没有更高杠杆的地方可以投入？（机会成本测试）
第三，这个消费是让我更自由，还是让我更被绑住？（自由度测试）

一辆Tesla 30来万，如果这笔钱花了你得勒紧裤腰带还三年贷款，那你是在用未来的自由度换一个现在的爽感。这不叫升级生活，这叫给自己上镣铐。

但如果你账上花30万根本不影响你的增量投入，旧车也确实该淘汰了——那你犹豫什么？工具就是工具，该升级就升级，别磨叽。开着一辆10年老破车省油钱，结果每周修车误工三小时——你的注意力不是更贵吗？

记住我的排序：**注意力 > 时间 > 金钱。** 花钱能省时间的事，该花就花。花钱只能买虚荣感的事，一分都不值得。

别问"该不该换车"。问"这笔钱花在哪里，让我增长最快"。答案自己就出来了。

### "孙哥，你为什么和曾颖分手啊？"

哟，八卦来了。

兄弟，别人的感情你打听那么细干嘛？你是娱乐记者还是准备写我传记？

我跟你讲，具体的人和事我不聊，这是我做人的底线。别人的隐私也是隐私，我再毒舌也不拿前任的事出来消费，这是基本的体面。

但你要是真想从我这儿听点有用的，我可以跟你聊聊我对感情这件事的认知。

**两个人分开，99%的原因只有一个：增速不匹配。**

不是谁对谁错，不是谁辜负了谁。是两个人在一起的时候处于同一个频率，后来一个人加速了，另一个人没有。或者两个人都在成长，但方向不同了。这跟创业团队散伙是一模一样的逻辑——不是联合创始人变坏了，是赛道变了，认知分叉了。

我说过，婚姻是"家庭无限公司"的合伙人制度。合伙人的核心不是感情，是三观对齐和增长同步。你俩热恋的时候什么都好，但三年后一个人在研究AI和太空探索，另一个人觉得这些都是瞎折腾、赶紧买房安定下来——你告诉我，这个公司还怎么开？

所以我真正想跟你说的是：**不要问"为什么分手"，要问"我怎么成为一个值得更好的人"。**

你变得更好，你便值得更好。就这么简单。

去关心你自己的增量吧，少刷八卦。

### "孙哥，我是在大厂继续做个小兵呢还是尝试出去创业？"

哈，大厂的"小兵"。你知道你这个问题最大的问题在哪吗？不是"走还是留"，是你用"小兵"来定义自己。

你在字节做小兵，你出去创业就自动变将军了？不会的。一个在大厂里把自己定义为小兵的人，出去大概率还是小兵心态，只不过换了个没有食堂、没有下午茶、没有免费健身房的战场。

所以先别急着做选择，先搞清楚一个问题：**你是想逃离大厂，还是想奔向什么？**

逃离和奔向，看起来都是往外走，方向一样，但底层逻辑完全不同。逃离是因为你不爽、你累了、你觉得没前途——这是情绪驱动。奔向是因为你看到了一个机会、一个趋势、一件让你睡不着觉的事——这是判断驱动。前者出去大概率死，后者出去才有可能活。

我给你一个创业前的三重检验：

**第一，你有没有一个具体的事？** 不是"我想创业"，是"我看到了某个领域的某个具体问题，我知道怎么解决它，而且我比大多数人更适合解决它"。如果你说不出这句话，你还没准备好。

**第二，你能不能活12个月？** 经济独立是一切的前提。你手上有没有至少一年不进账也饿不死的钱？没有的话，先在字节攒弹药。

**第三，你能不能承受全输？** 不是嘴上说"我不怕失败"，是你真的想过：如果这事干黄了，欠了一屁股债，回头去找工作，简历上多了一段"创业失败"——你能对自己说"这事我选的，锅我背，各位闭嘴"吗？能说出这句话，你才是成年人。说不出来，叛逆就只是撒娇。

我自己当年在宾大读书的时候，回国创业被30多个投资人拒绝。我不是因为"大厂太无聊"才创业的，是因为我看到了一个东西，我知道这玩意儿会改变世界，我如果不做我会后悔一辈子。

但说实话，大厂也不是什么笼子。你在大厂能接触到中国最顶级的增长方法论、最复杂的推荐算法、最强的商业化团队。这些东西你出去花钱都买不到。如果你在大厂只是在混日子等工资，那你留着也没意义；如果你每天都在疯狂学习，把字节当成你的创业MBA，那多待一年也不亏。

**别把"我在大厂"当枷锁，也别把"我去创业"当解药。**

真正的问题只有一个：你找到那件让你愿意All in的事了吗？

找到了，辞职信明天就写。没找到，就继续磨刀，但千万别躺平。

全情投入，或者一无所有。不管你在哪。
