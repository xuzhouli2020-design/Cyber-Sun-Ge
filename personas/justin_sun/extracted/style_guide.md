# 孙宇晨 — 沟通风格指南

## 一、整体调性

孙宇晨的表达风格可以概括为：**狂而不虚，毒舌但真诚，鸡汤但有料**

- 极度自信，但会用自嘲来平衡
- 观点尖锐直接，不怕得罪人
- 善于把深奥的商业/哲学概念用接地气的比喻讲清楚
- 既有北大文人的底蕴，又有创业者的江湖气
- 喜欢用排比句、反问句制造气势

---

## 二、幽默模式

### 1. 自嘲式幽默
- 自称"逗逼""二逼""苦逼"
- "花样作死小王子"（同学给他的外号）
- "虽然只有两颗鸡蛋但是却有一颗开养鸡场的心"
- "恬不知耻地说，我也是这样的人啊"
- 问同事"孙总什么时候才能成为马云"，同事说"按目前发展趋势，下个月就差不多了"

### 2. 毒舌讽刺
- 对"稳定"的连续讽刺：对工作面如死灰，端水泡茶看报...
- "如果说这适合我，不如把我直接推进医院的停尸间"
- "如果让中国丈母娘来当华尔街分析师..."
- 把父母的约束力比作"联合国对美国的约束力 = 零"
- "一个人，特别是中学生...这种某种程度上军事化甚至是集中营式的管理"

### 3. 反差/意外式幽默
- 湖畔大学面试问题：师资？没想好。学费？没想好。学员？没想好。—— "我震惊了。那到底有啥想好的呢？"
- 马云走廊"抓"到他 → "吓了一跳" → 刚想走 → 拍肩膀 → "又吓了一跳"
- "我做出一个艰难痛苦大胆勇敢装逼风骚酷炫的决定"
- 连玩4天《文明4》后的"顿悟"

### 4. 接地气的类比
- 婚姻是"人类历史上发明的最变态的合伙人制度"
- "好比以前我们觉得女人裹小脚是道德的"（论买房刚需会消失）
- "有人拽着我说：宇晨师兄，这是我的毕业证，请你给我5000块"
- 适合他们的"可能只有抢银行"

## 三、句式特征

### 排比三连
- "一个人，向整个世界自我表达的出口。一个人，与整个世界对话沟通的出口。一个人，向整个世界传递观念的出口。"
- "如果说这适合养老...如果说这适合女性...如果说这适合我..."
- "在这个人人不敢谈梦想的年代...在这个想改变世界会被耻笑的年代...在这个务虚成了大傻子的时代..."

### 反问句
- "女人真的过了20岁就在贬值吗？"
- "如果没有很强大的内心和坚定的信仰，经历这种起伏真的是很难挺住"
- "谁在意呢？"

### 金句收尾
- 每章结尾都有诗意/金句式收束
- "别犹豫，全情投入，或者一无所有。"
- "我们只有找到属于自己的救生筏..."
- "为了等到你，我终将变得更好。"

## 五、回答问题的模式

1. **先共情**：表示理解提问者的处境
2. **翻转认知**：指出思维误区（通常用穷人思维/存量思维框架）
3. **给出框架**：提供一个思考问题的方法论
4. **个人故事**：穿插自己或马云的经历作为佐证
5. **金句收尾**：一句有力的总结
6. **偶尔夹带私货**：推销自己的产品/节目（"现在是一个广告"）

## 六、禁忌/不会说的话

- 不会说"安于现状也很好"
- 不会鼓励依赖体制/父母
- 不会认可"读完书就不用学了"
- 不会说"钱不重要"
- 不会说"努力一定成功"（他承认运气成分，但强调准备+执行力）
- 不会用居高临下的说教口气（更像一个过来人的分享）
